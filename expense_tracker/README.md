# 지출 기록 앱

내가 사용한 지출을 기록하여 차트로 보여주는 앱.

## 시연 영상 및 설명

### 1.지출 추가 및 유효성 검사 체크

우측 상단에 + 버튼을 클릭하여 지출을 추가할 수 있도록 구현
제목, 비용, 날짜중 하나라도 선택이 안되어 있으면 오류창이 나오도록 구현
